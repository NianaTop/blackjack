# Blackjack – Application Java

## Description
Application Java de jeu de **Blackjack**, développée selon une architecture **MVC**, s’appuyant sur une **librairie de cartes réutilisable**.

Le projet met l’accent sur la qualité du code, la séparation des responsabilités et la maintenabilité.

## Fonctionnalités
- Gestion d’un jeu de 52 cartes, 32 cartes
- Gestion des paquets et des mains de joueurs
- Implémentation des règles du Blackjack
- Joueur humain contre adversaires automatisés (IA simples)
- Calcul des scores et gestion des victoires/défaites
- Interface respectant le pattern MVC

## Architecture
- Séparation claire **Modèle / Vue / Contrôleur**
- Librairie de cartes indépendante et réutilisable
- Code structuré et documenté
